Authful

An open-source API for Two Factor Authentication. By MongoHQ.

THE EASIEST WAY TO IMPLEMENT 2FA

After evaluating some of the popular two factor authentication services on the market, we noticed that they require users to download and use that service's mobile app. We believe users should have a choice which mobile app they use and that it should still be easy for developers to deliver that solution. Let's keep it open.

Features

 

Support for multiple mobile apps

 

SMS integration

 

Fallback numbers and recovery codes

For more information and documentation, check out the project on Github.
